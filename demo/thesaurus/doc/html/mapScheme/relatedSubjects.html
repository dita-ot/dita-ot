relatedSubjects

Specifies an associative relationship for the contained subjects.

Description

The type of association can be indicated by the title, by a reference to a subject topic that defines the association, or by specializing the relatedSubjects element to define the relationship. Such specializations can express roles by requiring specialized topics in specific positions. See also http://www.w3.org/TR/swbp-skos-core-guide/#secassociative.
