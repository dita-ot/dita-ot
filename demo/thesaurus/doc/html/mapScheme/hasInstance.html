hasInstance

Specifies that the contained hierarchy expresses INSTANCE-OF relationships between subjects.

Description

In an INSTANCE-OF hierarchy, the child subject is a specific entity or object and the parent subject is a type, kind, or class of entity or object. For example, New York City is a specific instance of a city.
