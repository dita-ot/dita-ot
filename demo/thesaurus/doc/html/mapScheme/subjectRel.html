subjectRel

Defines an association between subjects where the members of the association play different roles.

Description

You use the subjectRel element to specify a row in a subject relationship table. Each cell in the row contains one or more subjects that play a specific role within the association. You can specify the role indicated by the column using the subjectRelHeader element.
