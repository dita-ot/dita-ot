subjectdef

Within a scheme, refers to a topic that defines a formal subject.

Description

You can use the navtitle and locktitle attributes to specify a preferred label for the subject within the scheme that is different from the default preferred label specified in the subject topic. The subjectmeta element provides other ways of overriding the default definition of the subject for the scheme.
