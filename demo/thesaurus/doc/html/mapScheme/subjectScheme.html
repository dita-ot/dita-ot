subjectScheme

Organizes subjects into a taxonomy for processing or display.

Description

Like other DITA maps, the scheme organizes references to topics. In the case of the scheme, however, the references refer to topics that have been defined as formal subjects. The scheme can express hierarchical relationships between these subjects or associative relationships. See also http://www.w3.org/TR/swbp-skos-core-guide/#secscheme/
