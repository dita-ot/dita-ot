topicSubjectHeader

Specifies constraints on the subjects used in classifications.

Description

You use the topicSubjectHeader element to supply a header row for a topic classification table when you want to allow only specific kinds of subjects in each column. To specify a facet classification, each cell in the header row can identify a subject topic that must be the ancestor in some scheme for the subjects in the same column within the classification rows. For instance, if the cell within the header row specifies a User Action subject, the subjects in the column must be kinds of user actions.
