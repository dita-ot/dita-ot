subjectCell

Specifies subjects for a constrained classification of content topics.

Description

This cell can be empty in the topicSubjectHeader element but must identify the classified topics within the topicSubjectRow element.

Within the topicSubjectHeader element, the subjectCell element specifies a constraint on the classifying subjects in the column. For a facet classification, the subjectCell element specifies the top subject for a facet category.

Within the topicSubjectRow element, the subjectCell element specifies the subjects that classify the content topics in the topicCell element within the same row. If a header row specifies a constraint for the column, the subjects in the cell must conform to that constraint. For a facet classification, the subjects must appear within the hierarchy for the facet category in a scheme.

Use the topicsubject element if the subject is the primary subject for the content topics. Otherwise, use the subjectref element.

Note: Each content subject can only have one primary subject.

Contained by

topicSubjectHeader, topicSubjectRow

Contains

subjectref, topicsubject

Content model

( (topicsubject) (optional) then (subjectref) (any number))
