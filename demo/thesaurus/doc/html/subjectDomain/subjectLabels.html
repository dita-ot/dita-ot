subjectLabels

Contains the preferred label and synonyms used to name the subject.

Description

The subject labels also include icons used the represent the subject. You can override the subject labels within a scheme by supplying other labels in the scheme map. See also http://www.w3.org/TR/swbp-skos-core-guide/#seclabel.
