subjectDetail

A special section within a concept topic that defines the concept as a formal subject.

Description

The subject detail includes the preferred label for the subject, its synonyms, icons used to represent the subject, notes that specify the subject matter, and notes about the creation and maintenance of the definition. Once defined, a subject can be used to classify content.

See also http://www.w3.org/TR/swbp-skos-core-guide/#secconcept.
