Subject definition topic

You create a subject topic by providing a formal definition for a concept.

Description

The formal definition consists of a subjectDetail section that specifies the labels and icons for representing the subject and the notes defining the subject coverage or recording its modification.

Example

Here is an example of a definition of the Configuring subject:

<concept id="configuring">
    <title>Configuring</title>
    <shortdesc>You configure components to set up or refine your solution.</shortdesc>
    <conbody>
        <p>You don't have to get the best configuration the first time....</p>
        <subjectDetail>
            <subjectLabels>
                <altLabel>Setting up</altLabel>
            </subjectLabels>
            <scopeNote>Administrative actions performed after installation...</scopeNote>
        </subjectDetail>
    </conbody>
</concept>
  • altLabel
    An optional synonym for naming this subject.
  • altSymbol
    An alternative symbol used to identify the subject.
  • changeNote
    A listing of changes to the subject definition.
  • editorialNote
    Any comments for others who maintain the definition of the subject.
  • fullnoteref
    A cross reference to a topic with a more complete treatment than the subject note.
  • hiddenLabel
    An name that is sometimes applied to the subject but shouldn't be presented to the user.
  • historyNote
    A discussion of changes in the definition of this subject.
  • prefLabel
    The standard name for the subject.
  • prefSymbol
    A standard symbol used to represent this subject
  • scopeNote
    Defines the limits on what this subject covers.
  • subjectDefinition
    A concise verbal definition of the subject that can be displayed to the user.
  • subjectDetail
    A special section within a concept topic that defines the concept as a formal subject.
  • subjectLabels
    Contains the preferred label and synonyms used to name the subject.
  • subjectNote
    A miscellaneous public or private note about the subject definition.
