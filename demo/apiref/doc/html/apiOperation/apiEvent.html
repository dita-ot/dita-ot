apiEvent

The <apiEvent> element identifies an event action that is an instance of a classifier.

Description

An <apiEvent> can also raise a typed value such as a signal or a classifier such as an exception. In the <apiEventDef> element, the <state>, <keyword>, and <ph> elements are provided to support unexpected contingencies within the value to return.
