apiClassifierDef

The <apiClassifierDef> element represents the definition of the classifier.

Description

A class definition describes the rules by which objects behave. These objects are referred to as "instances" of that class. When a specialization does not provide a complete representation of the interface within a specialized <apiClassifierDef> element, you can use the <apiSyntax> element.
