apiRef module

Provides the structure and elements for non language specific API reference topic types.

Description

The primary purpose of the API item topic is to provide a basis for sharing elements among more specialized API item topics.

Example

<apiRef id="">
   <apiName></apiName>
   <shortdesc></shortdesc>
   <apiDetail></apiDetail>
</apiRef>
  • apiRef
    The <apiRef> element is the top-level element for generic API reference topic type.
