apiName

The <apiName> element provides the name of an application programming interface (API).

Description

The <apiName> element is contained not only by the <apiRef> element, but also by all specialized API topic types.

Example

<javaField id="IRXEAbstraction">
   <apiName>IID</apiName>
   <shortdesc>....</shortdesc>
</javaField>
<apiValue id="parameter">
   <apiName>theNameOPT</apiName>
   
</javaParameter>

<javaPackage>
	 <apiName>com.ibm.xtools.patterns.framework.uml2</apiName>
   <shortdesc>....</shortdesc>
<javaPackage>
<javaMethod>
	 <apiName>getPriority</apiName>
   <shortdesc>....</shortdesc>
<javaMethod>
