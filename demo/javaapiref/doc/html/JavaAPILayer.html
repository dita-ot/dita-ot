Java topic types

The Java API specialization provides a set of specific topic types that can describe a Java application programming interface (API) library. The Java language-specific API topic types are specialized from the generic API reference topic types. This specialization permits the sharing of the basic semantics and processing.

Description

The Java modules describe the Java topic types and their contained elements. For example, the javaPackage module describes all the elements necessary to describe a Java API package. The Java topic types provide language-specific topic types for documenting Java API references.

The Java topic types and the contained elements are described in the following modules:

  • javaPackage module
    Provides all the elements that describe the items in a Java API package.
  • javaClass module
    Defines a top-level container for all the elements that are necessary to describe an API class.
  • javaInterface module
    Defines all the elements that are necessary to describe an API interface.
  • javaMethod module
    Describes all the elements necessary for the basic structure of a Java API method.
  • javaField module
    Describes the elements necessary to show the name of the component fields and the short description of the field.
  • javaapi-domain module
    Provides the description of the elements that refer to the API items within Java API topics or other DITA topic types, such as concept or task.
  • javaAPIMap module
    Provides the relationships among a set of Java API topic types, and can be used to create a Table of Contents (TOC), aggregate Java API topics into a PDF document, or to create links between topics in output.
