javaInterfaceDef

The <javaInterfaceDef> element provides the interface modifier elements for the interface declaration.

Description

An interface declaration specifies a new named reference type as in the following Java syntax:

InterfaceDeclaration:
      InterfaceModifiersopt   interface Identifier   ExtendsInterfacesopt    InterfaceBody

The protected and private access modifiers pertain only to the member interfaces within the enclosed class declaration. The static access modifier pertains only to the member interfaces. The identifier in an interface declaration specifies the name of the interface. A compile error occurs if an interface has the same simple name as any of its enclosing classes or interfaces.
