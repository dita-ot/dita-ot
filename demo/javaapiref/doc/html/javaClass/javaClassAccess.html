javaClassAccess

The <javaClassAccess> element provides the ClassModifiersopt for the declaration of the class access values.

Description

A class declaration can include the following modifiers: public, abstract, or final. Fields, methods, member class, member interface, and constructor declaration can include the access modifiers public, protected, or private.
