javaPackageRef

This element provides a Java package topic type as a link in a DITA map.

Description

A <javaPackageRef > can contain other <javaPackageRef > elements, allowing you to express navigation or table-of-contents hierarchies, as well as implying relationships between the containing <javaClassRef>, <javaInterface>, <javaException> and <javaError> elements and its children.

Example

Click on the following link to open the Java API topic type example in 
<javaPackageRef navtitle="Package com.ibm.xtools.patterns.framework.uml2"
              href="com.ibm.xtools.patterns.framework.uml2.dita" locktitle="yes">
     <javaClassRef navtitle="UmlnotationPlugin.Implementation Class" href="UmlnotationPlugin.Implementation.dita">
     </javaClassRef>
     <javaClassRef navtitle="UMLDiagramResourceUti Class" href="UMLDiagramResourceUtil.dita">
     </javaClassRef>
</javaPackageRef>
