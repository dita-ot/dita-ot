cxxStructAbstract

The <cxxStructAbstract> element contains optional abstract information.

Description

The <cxxStructAbstract> element resides within the <cxxStructDefinition> element. The cxxStructAbstract element has no children but has the attributes name and value. name and value are both set to "abstract".
