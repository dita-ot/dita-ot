cxxEnumeratorAPIItemLocation

The <cxxEnumeratorAPIItemLocation> element is defined within <cxxEnumerator>. <cxxEnumeratorAPIItemLocation> specifies the location of the API.

Description

The <cxxEnumeratorAPIItemLocation> element must contain the location of the API declaration using <cxxEnumeratorDeclarationFile> and <cxxEnumeratorDeclarationFileLine>.

Example

<cxxEnumeratorAPIItemLocation>
				<cxxEnumeratorDeclarationFile name="filePath" value="D:/src/include/headerfile.h"/>
				<cxxEnumeratorDeclarationFileLine name="lineNumber" value="775"/>
			</cxxEnumeratorAPIItemLocation>
