cxxAPIMap module

Provides the relationships among a set of C++ API topic types, and can be used to create a Table of Contents (TOC), aggregate C++ API topics into a PDF document, or to create links between topics in output.

Description

Maps provide a way to express relationships among a set of C++ API topic types in a single common format for the C++ API reference that can be used for different outputs. The cxxAPIMap module organizes your DITA files into a hierarchy. It can also create relationship links between your topics. Use <cxxAPIMap> to organize your work, and to create and edit C++ API DITA files.

The cxxAPIMap topic contains the following elements:

  • cxxAPIMap
    The <cxxAPIMap> element provides the name of the C++ API Reference and all the elements necessary to describe the package, or set of packages, and its components.
  • cxxClassRef
    This element provides a C++ class topic type as a link in a DITA map.
  • cxxDefineRef
    This element provides a C++ define topic type as a link in a DITA map.
  • cxxEnumerationRef
    This element provides a C++ enumeration topic type as a link in a DITA map.
  • cxxFileRef
    This element provides a C++ file topic type as a link in a DITA map.
  • cxxFunctionRef
    This element provides a C++ function topic type as a link in a DITA map.
  • cxxStructRef
    This element provides a C++ struct topic type as a link in a DITA map.
  • cxxTypedefRef
    This element provides a C++ typedef topic type as a link in a DITA map.
  • cxxUnionRef
    This element provides a C++ union topic type as a link in a DITA map.
  • cxxVariableRef
    This element provides a C++ variable topic type as a link in a DITA map.
