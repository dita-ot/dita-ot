cxxFile module

The cxxFile module defines a top-level container for all the elements necessary to describe the global APIs of a component

Description

The <cxxFile> topic type contains elements necessary to show the name of the file, and also the global variables, functions, defines, enumerations and typedefs contained within.

The cxxFile topic contains the following elements:

  • cxxFile
    The <cxxFile> element contains all the global APIs of a component.
  • cxxFileAPIItemLocation
    The <cxxFileAPIItemLocation> element is a container for the element that describes the location of the file containing the global APIs.
  • cxxFileDeclarationFile
    The <cxxFileDeclarationFile> element defines the location of the declaration file described in this document.
