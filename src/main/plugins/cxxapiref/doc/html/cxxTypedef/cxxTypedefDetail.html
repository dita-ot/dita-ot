cxxTypedefDetail

The <cxxTypedefDetail> element provides detailed information about the structure and behaviors of the C++ typedef.

Description

The <cxxTypedefDetail> element is defined within <cxxTypedef>. In <cxxTypedefDetail> the <example> and <section> elements are provided to support the current specific C++ API reference.
