cxxStructBaseUnion

The <cxxStructBaseUnion> element declares inheritance from a base union in a struct derivation.

Description

Used within a <cxxStructDerivation> element to declare inheritance from a base union.

Example

<cxxStructDerivation>
    <cxxStructDerivationAccessSpecifier value="public"/>
    <cxxStructBaseUnion href="union_base">Base</cxxStructBaseUnion>
</cxxStructDerivation>
