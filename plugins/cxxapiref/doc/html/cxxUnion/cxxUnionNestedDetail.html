cxxUnionNestedDetail

The <cxxUnionNestedDetail> element provides details about the nested members of a C++ union.

Description

In the <cxxUnionNestedDetail> element, the <cxxUnionNestedClass>, <cxxUnionNestedUnion> and <cxxUnionNestedStruct> elements are provided to indicate all members that can be nested by a C++ union. <cxxUnionNestedDetail> must have at least one of these child elements and each one can occur multiple times.

Example

<cxxUnionNested>
    <cxxUnionNestedDetail>
        <cxxUnionNestedStruct href="struct_comms.xml#struct_comms_package">CommsFW::TCFLegacyMessagePacker::TUnion::TLegacyPackage</cxxUnionNestedStruct>
        <cxxUnionNestedStruct href="struct_comms.xml#struct_comms_package">CommsFW::TCFLegacyMessagePacker::TUnion::TPackage</cxxUnionNestedStruct>
    </cxxUnionNestedDetail>
</cxxUnionNested>
