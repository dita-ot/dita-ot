cxxEnumeratorPrototype

Provides a "visual" representation of the enumerator declaration.

Description

This element provides a "visual" representation of the enumerator declaration as PCDATA. The are no semantics applied to the content but the intention is that the content should look as similar as possible to the enumerator declaration in source code.
