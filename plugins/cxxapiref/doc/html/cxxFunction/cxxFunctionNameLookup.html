cxxFunctionNameLookup

Provides a unique identifier of the function.

Description

This element provides a way of uniquely identifying a function in a body of DITA XML that describes a body of C++ source code.

Implementations are free to write what they want in this element's PCDATA but a "good" implementation would place some premium on the following:
  • The text would be unique within a set of DITA XML files.
  • The text would be human readable.
  • The text would generally follow the lookup techniques use in the C++ standard (ISO/IEC 14882:2003(E)).
  • Any encoding used would be reversible.
