cxxFunctionDefinitionFile

Identifies the path to the file that contains the definition of the function.

Description

This identifies the path to the file that contains the definition of the funcition. This has two attributes; name (always has a value "filePath") and value.

Example

<cxxFunctionDefinitionFile name="filePath" value="inc/a.h"/>
