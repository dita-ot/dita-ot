cxxClassBaseClass

The <cxxClassBaseClass> element declares inheritance from a base class in a class derivation.

Description

Used within a <cxxClassDerivation> element to declare inheritance from a base class.

Example

<cxxClassDerivation>
    <cxxClassDerivationAccessSpecifier value="public"/>
    <cxxClassBaseClass href="class_base">Base</cxxClassBaseClass>
</cxxClassDerivation>
