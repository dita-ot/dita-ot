cxxClassDerivationAccessSpecifier

The <cxxClassDerivationAccessSpecifier> element indicates the access specifier used when a class inherits from another class.

Description

The <cxxClassDerivationAccessSpecifier> element is contained within a <cxxClassDerivation> element. The value attribute is set to public, protected or private to describe the access the class has on the members of the base that is being derived from. The base referered to by the <cxxClassDerivationAccessSpecifier> element is indicated by using either a <cxxClassBaseClass>, <cxxClassBaseStruct> or <cxxClassBaseUnion> element as a sibling of <cxxClassDerivationAccessSpecifier>.

Example

<cxxClassDerivation>
	<cxxClassDerivationAccessSpecifier value="public"/>
	<cxxClassBaseClass href="class_base">Base</cxxClassBaseClass>
</cxxClassDerivation>
