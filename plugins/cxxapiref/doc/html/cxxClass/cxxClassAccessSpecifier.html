cxxClassAccessSpecifier

The <cxxClassAccessSpecifier> element describes the level of access the class allows.

Description

The <cxxClassAccessSpecifier> element does not have any children. <cxxClassAccessSpecifier> uses the attribute value to describe the level of access a class has. value can be set to "public", "protected" or "private"; the default is "public".

Example

<cxxClassAccessSpecifier value="public"/>
