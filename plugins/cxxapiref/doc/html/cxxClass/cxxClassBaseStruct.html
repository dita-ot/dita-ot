cxxClassBaseStruct

The <cxxClassBaseStruct> element declares inheritance from a base struct in a class derivation.

Description

Used within a <cxxClassDerivation> element to declare inheritance from a base struct.

Example

<cxxClassDerivation>
    <cxxClassDerivationAccessSpecifier value="public"/>
    <cxxClassBaseStruct href="struct_base">Base</cxxClassBaseStruct>
</cxxClassDerivation>
